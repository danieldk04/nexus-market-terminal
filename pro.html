Upload of plak hier je Trade Republic- en BUX-exports. De pagina leest ze direct in je browser uit (niets wordt verstuurd) en toont een preview van je posities, gemiddelde kostprijs en huidige waarde — zodat je kan controleren of alles klopt vóórdat je het definitief verwerkt.

Let op: deze site is statisch (GitHub Pages) en heeft geen eigen server — het dashboard (Portfolio-tab, historiek) wordt elke ochtend automatisch bijgewerkt door een GitHub Action die de CSV's leest uit GitHub Secrets. Deze Import-tab kan die stap niet overslaan (dat zou credentials vereisen die ik nooit namens jou mag invoeren), maar wél het kopiëren foutloos maken: parse → controleer de preview → kopieer → plak in de juiste secret → run de workflow.
